Chrome DevTools Console Methods

Open your browser console (F12 or Ctrl+Shift+J) to see the magic happen!

Styling console.log()

Add custom CSS styles to your console messages

console.log('%c Hello World', 'color: blue; font-size: 20px;');

Console Message Types

Different log types for different purposes

console.log('Standard log message');
console.warn('This is a warning');
console.error('This is an error');
console.info('This is an info message');

console.assert()

Logs a message only if the assertion fails

console.assert(1 === 2, 'Assertion failed: 1 is not equal to 2');

console.dir()

Displays a DOM element as a JavaScript object

const el = document.querySelector('h1');
console.dir(el);

Grouping Console Logs

Group related messages together

console.group('User Info');
console.log('Name: John Doe');
console.log('Age: 30');
console.groupEnd();

Measuring Performance

Measure execution time with console.time()

console.time('Loop');
for (let i = 0; i < 1000000; i++) {}
console.timeEnd('Loop');

Counting Function Calls

Track how many times a function is called

function greet() {
  console.count('Greet called');
  return 'Hello!';
}
// Click multiple times to see the counter increase

Open Chrome DevTools to see the results (F12 or Ctrl+Shift+J)
