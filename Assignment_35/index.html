Array Methods

.some()

Checks if at least one element passes the test

Example: Check if any person is an adult

const isAdult = people.some(person => 
  (new Date()).getFullYear() - person.year >= 18);

.filter()

Creates a new array with elements that pass the test

Example: Find all adults

const allAdults = people.filter(person => 
  (new Date()).getFullYear() - person.year >= 18);

Example: Find comment with ID 523423

const specificComment = comments.filter(
  comment => comment.id === 523423);

.find()

Returns the first element that passes the test

Example: Find person named 'Wes'

const wes = people.find(
  person => person.name === 'Wes');

.map()

Creates a new array by transforming each element

Example: Get array of all names

const names = people.map(person => person.name);

.sort()

Sorts the elements of an array

Example: Sort people by age (youngest first)

const sortedByAge = [...people].sort(
  (a, b) => a.year - b.year);

.reduce()

Reduces array to a single value by applying a function

Example: Sum of all years

const yearSum = people.reduce(
  (total, person) => total + person.year, 0);
